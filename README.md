# MANUAL DE USUARIO - PANADERÍA DELICIA

## 1. DESCRIPCIÓN DE LA APP
Panadería Delicia es una aplicación móvil que permite explorar y seleccionar productos de panadería y pastelería, agregarlos a un carrito y preparar pedidos. Su interfaz está diseñada para ser rápida e intuitiva, facilitando la navegación y selección de productos.

## 2. REQUISITOS DEL SISTEMA
- Android 5.0 (Lollipop, API 21) o superior.  
- Espacio libre aproximado: 50 MB.  
- Conexión a Internet (si la app carga datos desde un servidor).  
- Permisos: Acceso a Internet; almacenamiento solo si se guarda caché de imágenes.

## 3. INSTALACIÓN
### Instalación mediante APK
1. Descargar `PanaderiaDelicia.apk`.  
2. Abrir el archivo desde la carpeta Descargas.  
3. Activar "Permitir instalaciones de fuentes desconocidas" si el sistema lo solicita.  
4. Pulsar **Instalar** y esperar.  
5. Abrir la app desde el menú de aplicaciones.

## 4. FUNCIONALIDADES

> 4.1 Catálogo de Productos  
**Descripción:** Lista todos los productos disponibles con imagen, nombre y precio.  
**Pasos:** Inicio → desplázate → toca un producto para ver detalles.

> 4.2 Detalle del Producto  
**Descripción:** Muestra imagen, descripción, precio y botón **Agregar al carrito**.  
**Pasos:** Desde el catálogo, seleccionar producto → usar botones de cantidad → Agregar.

> 4.3 Carrito de Compras  
**Descripción:** Visualiza productos seleccionados, permite cambiar cantidades y eliminar.  
**Pasos:** Tocar ícono del carrito → revisar items → ajustar cantidades → proceder.

> 4.4 Navegación por la app  
**Descripción:** Barra inferior con acceso a Inicio, Catálogo, Carrito y Perfil/Ajustes.  
**Pasos:** Tocar el icono deseado para ir a la sección correspondiente.

## 5. PREGUNTAS FRECUENTES (FAQ)
**La app no se instala.**  
- Habilita instalaciones de fuentes desconocidas; verifica espacio.

**La app falla o no abre.**  
- Reinicia el dispositivo; reinstala la app si persiste.

**No carga el catálogo.**  
- Revisa la conexión a Internet; borra caché de la app.

**Los botones no responden.**  
- Reinicia la app; revisa actualizaciones en Play Store (si aplica).
